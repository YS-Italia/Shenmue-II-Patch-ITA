## Come Installare la Patch

Segui questi passaggi per inserire i file tradotti nel gioco:

1. **Trova la cartella di Shenmue II:**
   * Apri **Steam** e vai nella tua Libreria.
   * Fai clic destro su **Shenmue I & II** -> **Gestisci** -> **Sfoglia i file locali**.
   * Nella finestra che si apre, naviga fino a questo percorso esatto:
     `SteamLibrary\steamapps\common\SMLaunch\sm2\archives\dx11\data`
     
2. **Fai un backup (Consigliato):**
   * Prima di toccare qualsiasi cosa, fai una copia di sicurezza della cartella `data` originale. Se qualcosa si rompe, ti basta rimettere quella.

3. **Incolla la patch:**
   * Scarica ed estrai lo `.zip` della patch.
   * Prendi i file estratti e **trascinali dentro la cartella `data` che abbiamo appena aperto**.
   * Se stai aggiornando la patch con una release futura, quando il sistema ti chiederà di sovrascrivere i file esistenti, clicca su **Sì / Sostituisci**.

4. **Avvia il gioco:**
   * Fai partire Shenmue II da Steam. 
   * Assicurati che la lingua dei testi nelle opzioni sia impostata su **Italiano** (la patch va a sostituire i file della lingua inglese e tutti i suoi riferimenti, quindi nelle impostazioni vedrai la lingua italiana).
